Release Notes

Release Notes

Adreno Profiler 3.8

  • Added Global GPU metrics grapher, allowing app developers to view system-wide GPU metrics
  • Added ability to view time spent in render pipeline stages per metric in Realtime Grapher
  • Added new grapher user interface features, allowing for more customization and enhanced metric viewing
  • Added support for OpenCL 1.2
  • Added OpenCL features:
    • Support for more than one context
    • Extended cl_mem_flags to describe how the host accessess the data in a cl_mem object
    • clEnqueueFillBuffer and clEnqueueFillImage support filling a buffer with a pattern or an image with a color
    • Add CL_MAP_WRITE_INVALIDATE_REGION to cl_map_flags
    • Support 1D image, 1D image from a buffer object, 1D image array and 2D image arrays
    • clCreateImage to create an image object
    • Support for clGetKernelArgInfo
    • Additional queries to get the number of kernels and kernel names in a program have been added to clGetProgramInfo
    • Additional queries to get the compile and link status and options have been added to clGetProgramBuildInfo
    • Support for clEnqueueMarkerWithWaitList and clEnqueueBarrierWithWaitList
    • Add options to control the OpenCL C version with Kernel Analyzer
    • Add options to query kernel arguments with Kernel Analyzer
  • Added support for Adreno 306 GPU
  • OpenGL ES Client-side arrays are now marked with a performance warning
  • Fixed crashes when capturing a DirectX frame
  • Fixed problems profiling Windows Phone 8 applications using auto mipmap generation
  • Fixed memory leak in Windows version of Adreno Profiler
  • Fixed issue with CPU timeline in OpenCL Scrubber inaccuracy
  • Fixed crashes when trying to open a saved session with some applications
  • Fixed binning not being reported per rendertarget
  • Fixed application crashes when trying to profiler some OpenGL ES 3.0 applications
  • Fixed issues capturing a frame with some OpenGL ES 3.0 applications on Linux
  • Fixed miscellaneous user interface issues

    Adreno Profiler 3.7

  • Adreno Profiler distributions now available for the Windows, Linux, and OS X desktop operating systems
  • Supports Adreno 4xx GPU's
  • New OpenGL ES features
    • Texture arrays
    • Transform feedback
    • Uniform buffers
  • Added ability to show depth stencil textures in the texture viewer
  • Added ability to show debug markers in OpenGL ES call list for apps using EXT_debug_marker
  • Added support for DirectX 11.1
  • Improved Profiler-device connectivity
  • Fixed various bugs causing Profiler to lose connection to the device or crash the app being profiled
  • Fixed shader "Send To Device" feature, allowing developers to modify a shader and upload it to the device
  • Fixed bug where overrides weren't working properly with cube map textures
  • Fixed problems capturing frames in some OpenGL ES and DirectX apps
  • Fixed OpenCL metrics:
  • Global Read Data
  • % Read L2 Hit/Miss
  • L2 Write Hit
  • % L1 Hit/Misses
  • L1 Hit/Miss Ratio
  • Number of Global Loads
  • Number Local Atomic
  • Fixed binning reporting
  • Fixed various OpenCL Scrubber issues with the Gantt view
  • Fixed error reporting the number of EFUs in shader stats
  • Fixed inaccuracy in reporting Pixel Write/Second OpenGL ES Scrubber metric
  • OpenCL Scrubber now captures 64-bit floating point buffer data
  • Fixed random bug with OpenGL ES render view's zoom-to-fit
  • Fixed OpenGL ES depth buffer view showing a white screen
  • Fixed support for full list of OpenGL ES 3.0 texture formats
  • Fixed many, many more bugs
