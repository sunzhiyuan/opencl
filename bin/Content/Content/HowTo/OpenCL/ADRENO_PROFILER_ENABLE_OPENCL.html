Adreno Profiler
ADRENO_PROFILER_ENABLE_OPENCL Reference

Summary:

Property used to enable and disable OpenCL profiling.

Usage:

To enable OpenCL profiler:



To disable OpenCL profiler:



Description:

When this property is initially set, non-blocking profiling is enabled by default which means that the application will continue execution without waiting for profiler to connect. If this behavior is desired please look at ADRENO_PROFILER_ENABLE_BLOCKING.

This property is needed as a "master toggle" to profile OpenCL applications and needs to be ALWAYS set if any kind of profiling is expected(either blocking or non-blocking).
