Scrubber GL :: API Calls

The "API Calls" tab in the Scrubber GL displays the OpenGLES API calls made prior to the selected draw call..





Basic usage:

• Capture a frame from the connected application, and select a render call in the call trace. The "API Calls" tab will list all API calls made between the selected render call and the previous render call.

• Hover the mouse over an API call to show extra information in a tooltip.

• Calls highlighted in red are considered redundant; they set state that has already been set by another call.

• Calls highlighted in blue are considered unused; they set state that will not be used in the frame, probably because a render feature is disabled. For instance, the blend function is unused in a frame that does not enable blending.

• Calls highlighted in orange are query calls, such as glGetString() and glIsTexture().
