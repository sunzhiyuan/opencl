Adreno Profiler
Requirements

Device Requirements

In general, Adreno Profiler works with all devices based on Qualcomm's application processors utilizing an Adreno 200 or newer GPU. This includes both commercial devices and development platforms. We support the following target operating systems:

Android: All versions of Snapdragon, as well as pre-Snapdragon app processors.

Windows RT/Windows Phone 8: Snapdragon S4 and newer

BlackBerry: BlackBerry 10


Additionally, the following APIs are supported:

OpenGLES v1.1 (Adreno 200+ only)

OpenGLES v2.0 (Adreno 200+ only)

OpenGLES v3.0 (Adreno 300+ only)

OpenCL (Adreno 300+ only)

DirectX 11 (Adreno 225+ only)

Software Requirements

Please ensure you have installed the latest drivers for your video card.

We have found it is sometimes necessary to install the latest reference drivers for your video card, directly from its manufacturer. If you are unable to view captured frames in a Scrubber with the drivers provided by your PC or laptop vendor, install the latest from ATI or NVIDIA.

If you are profiling DirectX applications, please ensure that you have DirectX Runtime installed on your PC.

Adreno Profiler runs under Microsoft Windows XP, Windows Vista, and Windows 7.

See Device Installation for device setup instructions.

Hardware Requirements

Adreno Profiler uses the desktop OpenGL ES emulator from the Adreno SDK to render the scrubber view for OpenGL ES applications. In order to render frames from OpenGL ES 2.0 applications, the emulator requires that the desktop GPU support OpenGL 2.0 or greater. For OpenGL ES 3.0 applicaitons, OpenGL 3.3 or greater is required.

The scrubber for DirectX applications renders directly using DirectX. The desktop GPU must support DirectX feature level 9_3.
