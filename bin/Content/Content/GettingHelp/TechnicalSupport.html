Getting Assistance

Before making a support request, please be sure your problem is not addressed by this documentation, and that you have installed the latest version of Adreno Profiler and the latest version of reference video drivers from your video card manufacturer.


Please contact us on the Mobile Gaming & Graphics Optimization (Adreno™) forums to report problems or ask questions about Adreno Profiler.


If reporting a bug, please provide as much information as possible, including:

Adreno Profiler application version, as listed in the about box

Mobile device model and build version

Detailed steps to reproduce the problem

A screen shot of the problem

Binaries of the embedded content used, if possible

A "Microsoft System Information" file

• First, make sure your mobile device is connected to the PC

• Open the "Run..." dialog from the Start Menu, and launch "msinfo32.exe"

• Within the "System Information" application, choose "Save..." from the File menu

• Attach the resulting NFO file to your email
