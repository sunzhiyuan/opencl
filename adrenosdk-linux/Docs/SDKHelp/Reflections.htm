Reflections

This sample demonstrates how to create dynamic reflections using a cubemap and framebuffer objects (FBOs).

Details

Framebuffer objects allow for fast, efficient rendering to offscreen surfaces. These surfaces can be bound as textures, allowing render-to-texture effects without having to copy the framebuffer back into system memory, and then sending the image again as texture data.

This example shows how to create and bind framebuffer objects to a cubemap which is then used to create dynamic reflections.

At initialization, this sample creates an empty cubemap and attaches 6 framebuffer objects to it. Then, at the beginning of each frame, an environment map for the center object is created by rendering the scene 6 times from the object's point of view, once for each direction, using a 90 degree field of view.

The scene is then rendered normally, with the center object using the environment map and texture coordinates dynamically generated in the vertex shader.
