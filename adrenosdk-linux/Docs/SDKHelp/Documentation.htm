Documentation

This section holds documentation for items relevant to the Adreno SDK. Below is a list of the documentation that is included.

Documentation
Adreno Developer Guide The Adreno Developer Guide provides useful information for developing and optimizing applications on Qualcomm's Snapdragon mobile platforms which use the Adreno GPU (Graphics Processing Unit).
OpenGL ES 3.0 Quick Reference Guide This document is a quick reference guide to explain how to utilize OpenGL ES 3.0 APIs within your Android applications running on Snapdragon processors with integrated Adreno 3xx GPUs.
Adreno DirectX Developer Guide The Adreno DirectX Developer Guide provides information for developing and optimizing DirectX applications for Windows RT on Snapdragon S4-based mobile platforms that include the Adreno series GPUs.
Shaders Best Practices A brief overview of best practices when writing shaders on Adreno hardware.
OpenCL General Programming Guidelines The Snapdragon OpenCL Software Development Guide explains how to develop efficient OpenCL applications that target one or both of the Adreno 3xx GPU and Krait CPU with references pointing to the Snapdragon 600 platform as an example.
Adreno SDK Help The Adreno SDK Help provides documentation on best practices as well as information about all of the samples, demos, and tools.
Emulator User Guide Qualcomm's OpenGL ES 3.0 Emulator is designed to emulate graphics functionality on mobile Qualcomm hardware. The purpose of this emulator is to aid developers in their design of OpenGL ES 3.0 games and applications.
Installation and Usage Information for Cross Platform installation and usage of this Browser.
