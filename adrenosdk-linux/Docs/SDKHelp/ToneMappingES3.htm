Tone Mapping

This sample demonstrates how to implement tone mapping in order to approximate the appearance of high dynamic range within a medium that has a more limited dynamic range.

OpenGL ES 3 Features

  • GL_DEPTH24_STENCIL8
  • GL_UNSIGNED_INT_24_8

Controls

KeyAction
*Change orientation
0Toggle help
1Decrease bright pass threshold
1Increase bright pass threshold
3Decrease bright pass offset
4Increase bright pass offset
8Toggle rotation
