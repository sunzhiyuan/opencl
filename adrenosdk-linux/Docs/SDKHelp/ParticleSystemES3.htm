Point Sprites

This sample demonstrates how to set up point sprites in OpenGL ES 3.0 by implementing a particle system.

OpenGL ES 3 Features

  • glBindBufferBase
  • GL_RASTERIZER_DISCARD
  • GL_TRANSFORM_FEEDBACK_BUFFER

Controls

KeyAction
*Change orientation
0Toggle help
1Decrease X Factor
2Increase X Factor
3Decrease Y Factor
4Increase Y Factor
5Decrease Speed
6Increase Speed
7Toggle Blend Mode
8Toggle Transform Feedback vs CPU Particles
