System Requirements

For this release of the SDK, the following requirements must be met.
  1. Graphics Card - Radeon X1300 or higher is recommended.
  2. Operating System - Windows XP with Service Pack 2 and latest security updates
  3. Development Environment - Visual Studio 2005
  4. Drivers - For AMD graphics cards, download the latest Catalyst driver from AMD's web site : http://www.ati.com
  5. Microsoft .Net Framework 2.0 - Needed to run the SDK Browser. Download from : http://www.microsoft.com/downloads/details.aspx?FamilyID=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=en

Known Issues

  1. Currently, the SDK runs only on a Windows XP machine with Radeon X1300 or higher graphics card.
  2. ImageEffects sample takes about 15 sec. to initialize. This time is spent generating the 3D textures required for the image effects.
  3. The MotionBlur sample's performance on Adreno™ hardware cannot currently be approximated.
  4. Currently, the SDK Installer has no way of checking if the user's machine has a valid graphics card or not. This will be added in the future versions.
  5. The SDK framework code currently supports 32 bit displays only. Additional depths will be added in the future.
  6. In order to remove a single installed component, please select 'modify' at the time of the uninstall. This will be improved in the next version.
  7. In the case that an error occurs while uninstalling the SDK, you can force the SDK to re-install on top of the old one by first running REGEDIT.EXE and searching for "Adreno", then deleting the first directory that comes up.
