Samples

The following lists all the samples that are currently shipped in this release.

All OpenGL ES samples have multiple help screens that list the controls that the samples accept. Press 0 to navigate through these screens. By pressing the * key, you can also change the orientation of these samples between portrait and landscape.

Samples
OpenGL ES 2.0 Samples The Adreno Developer Guide provides useful information for developing and optimizing applications on Qualcomm's Snapdragon mobile platforms which use the Adreno GPU (Graphics Processing Unit).
OpenGL ES 3.0 Samples This document is a quick reference guide to explain how to utilize OpenGL ES 3.0 APIs within your Android applications running on Snapdragon processors with integrated Adreno 3xx GPUs.
OpenGL ES 3.1 Samples The Adreno DirectX Developer Guide provides information for developing and optimizing DirectX applications for Windows RT on Snapdragon S4-based mobile platforms that include the Adreno series GPUs.
OpenCL Samples A brief overview of best practices when writing shaders on Adreno hardware.
Advanced Input Samples The Snapdragon OpenCL Software Development Guide explains how to develop efficient OpenCL applications that target one or both of the Adreno 3xx GPU and Krait CPU with references pointing to the Snapdragon 600 platform as an example.
OpenSLES 1.0 Samples The Adreno SDK Help provides documentation on best practices as well as information about all of the samples, demos, and tools.
