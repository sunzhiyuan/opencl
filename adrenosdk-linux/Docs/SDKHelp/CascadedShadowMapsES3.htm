Cascaded Shadow Maps

This sample demonstrates how to implement cascaded shadow maps in OpenGL ES 3.0. Cascaded shadow maps is a technique that extends regular shadow map implementations.

OpenGL ES 3 Features

  • glDrawBuffers
  • glTexture3D
  • glFrameBufferTextureLayer

Controls

KeyAction
*Change orientation
0Toggle help
1Next Model
2Rotate Sun CW about Z axis
3Rotate Sun CCW about Z axis
4Rotate Sun CW about X axis
5Rotate Sun CCW about X axis
6Change Shadow Filter
7Toggle Depth Buffer
8Change Cascade Split Method
