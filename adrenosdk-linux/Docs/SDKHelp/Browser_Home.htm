Welcome to the Adreno SDK Browser

This is version 1.0 of the new Adreno SDK Browser. Please read this document in order to become familiar with it's usage.

The Browser is built using Gtk#. Gtk# is a Graphical User Interface Toolkit for mono and .Net. The project binds the Gtk+ toolkit and assorted GNOME libraries, enabling fully native graphical Gnome application development using Mono and the .Net development frameworks. You can read more about GtkSharp here: www.mono-project.com/GtkSharp

There are several relevant Mono tools for the Browser:
MRE - Mono Runtime Environment: www.mono-project.com/Mono:Runtime
MDK - Mono Development Kit MonoDevelop - A cross-platform IDE primarily designed for C#/.Net: www.monodevelop.com

In order to run the Browser, only the MRE and the MDK are required. The Browser was developed with MonoDevelop but you will not need that package to use the Browser.

Usage

The Browser interface was redesigned with simplicity in mind. The Tree structure on the left hand side will allow you to navigate through all the included Samples, Tutorials, Tools, Libraries, and Documentation. As shown below, many items will have an image along with a short description in the right hand pane.



Toolbar

The Toolbar buttons at the top provide the various features of the Browser. Features exist depending on the item selected. From left to right, the button's functionality is:

NameImageDescription
DescriptionShow the default screen which is generally a description of the item.
RunRun the executable of the item selected.
CodeShow the code directory of the item selected.
DocsShow some detailed documentation of the item.
Build AndroidBuild the selection for Android devices.
Install AndroidBuild the selection for Android devices and install it over ADB.

The Toolbar buttons will only be available if the selection has the corresponding feature. For example, if an item from the Documentation section is selected, you will not be able to click the "Install To Android" button because documentation doesn't have that ability. Some options may be unavailable on certain Operating Systems.

In addition to the Toolbar Buttons, the commands have "hotkeys". The mappings are as follows:

NameImageKey
Descriptionh
Runr
Codec
Docsd
Build Androidb
Install Androida
