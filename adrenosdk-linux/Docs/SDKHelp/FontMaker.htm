Font Maker

This tool reads a font file and creates the necessary assets to display text on the screen during runtime.

Details

The Font Maker creates a bitmap of the font glyphs in TGA format that are laid out in a rectangular array. It also creates a font file containing spacing information and texture coordinates. The spacing information allows non-uniform letter spacing. The TGA file is referred to as the “bitmap” and the font file is referred to the “spacing” file below.

Since this tool writes out the bitmap in TGA format, its very easy to edit this file. This is useful for embedding special characters called wingdings into the font. For example, an image of a specific key on the gamepad could be included.

Usage

The syntax to run the tool from command line is:

FontMaker.exe <Font File> -b

Where Font File is the name of the font such as Tuffy 16.

This will create the bitmap and the spacing files: Tuffy_16.tga and Tuffy_16.font.

For use in the SDK samples these two files must be copied to the “Samples\Media\Font” directory. To import a font into a project, add the following line to the sample’s XML file:

<FontFileName = "..\..\Media\Fonts\Tuffy16.xml"/>
