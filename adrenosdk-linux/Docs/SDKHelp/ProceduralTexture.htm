ProceduralTexture

This sample demonstrates how to create a procedural texture.

Details

Typically applications make use of texture mapping in order to increase the realism of a 3D scene. On embedded devices resources are very limited, and as a result textures need to be small and heavily compressed in order to be useful. An alternative approach to using texture mapping to create realism is to assign procedural textures which are created on the fly.

This sample showcases a variety of different procedural textures. The first of which is a checker pattern whose frequency can be manipulated to control the number of boxes that make up the pattern. The second procedural texture uses a 3D noise texture to create a procedural wood texture whose frequency can also be manipulated. The third procedural texture also makes use of a 3D texture to create a marble-like pattern whose turbulence scale factor can be controlled by the user. Finally the fourth and final procedural texture once again uses a 3D texture, but this time to generate a cloudy texture whose brightness changes over time.
