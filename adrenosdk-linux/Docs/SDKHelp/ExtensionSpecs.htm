OpenGL ES 2.0 Extension Specs

The OpenGL ES API registry contains the latest specifications for the extensions that have been accepted. They can be found at : http://www.khronos.org/registry/gles/.

Below, is a list of supported extensions for easy reference, that have been accepted by the Khronos registry.

  • OES_compressed_ETC1_RGB8_texture
  • OES_depth24
  • OES_depth32
  • OES_element_index_uint
  • OES_fragment_precision_high
  • OES_get_program_binary
  • OES_rgb8_rgba8
  • OES_texture_3D
  • OES_texture_float
  • OES_texture_npot
  • OES_texture_float_linear
  • OES_texture_half_float_linear
  • OES_vertex_half_float
