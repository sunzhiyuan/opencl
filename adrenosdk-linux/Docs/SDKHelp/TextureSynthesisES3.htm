Texture Synthesis

This sample demonstrates how to generate textures on the fly for detailed terrain scenes with texture atlases.

OpenGL ES 3 Features

  • GL_COLOR_ATTACHMENTn
  • GL_RG16UI
  • GL_RG_INTEGER
  • GL_RGBA8
  • glDrawBuffers
  • glMapBufferRange
  • GL_MAP_WRITE_BIT
  • glUnmapBuffer
