PC2Android

PC2Android is a console application that allows the Snapdragon framework samples to be controlled via the numeric keys and mouse of the development PC. The Android sample must be started first before starting PC2Android.

To use, please ensure that the Fluid device is connected via usb. You must also make sure that you have your Android ADB environment set up properly (this needs to be done in order to run the SDK samples on Android).

Next, you must deploy one of the Snapdragon samples. As soon as the sample is running, deploy PC2Android. If the connection works, you will see the console application look like the above image. You can now start using the numeric keys or mouse to work on the samples as if they were on the PC.

There are a few known issues with this program. You cannot run more then one sample at the same time. The Snapdragon samples on Android get suspended when they are closed via the home button. Only the first sample will be getting the input from PC2Android, so if you start another sample with the first application in suspended state, PC2Android will not work. This will be fixed in future releases of the tool. As well, the Android sample must be run first (it is considered the 'server' in the connection) before the PC. This will also be potentially fixed in the future so that it doesn't matter which order the sample and PC2Android are run in.

The full source for PC2Android is made available to developers, as well as the server side code in the Snapdragon sample framework.
