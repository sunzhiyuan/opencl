Adreno Texture Tool Suite

Overview

The Adreno Texture Tool Suite allows one to visually compress textures into formats supported by the Adreno 2xx graphics hardware. It also allows for setting up projects that allow for compressing multiple textures and embedding compression to take place in an art pipeline via a command-line tool.

Features

The Adreno Texture Tool Suite offers the following features:
  • Visually compare different compression types
  • Zoom into textures for high level of detail compression comparison
  • Run a ‘best fit’ compression alalysis based on either texture size or quality
  • Visualze each color channel seperately, including alpha (if available)
  • Have multiple workspaces open at the same time
  • Save texture workspaces that can be processed within an art pipeline
  • Ability to generate mip-maps in texture
  • Saves files out to open-standard .ktx and .dds format
  • Includes a Photoshop plug-in (32 and 64 bit) allowing for artists to load compressed formats

For more information, please see documentation within the tools directory.
