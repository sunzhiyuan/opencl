VarianceShadowMap

This sample demonstrates how to create projective shadows.

Details

Variance shadow mapping is an extension upon traditional shadow mapping that attempts to add support for soft-shadowing. The basic idea is that instead of having a shadow map of depth values, we also store depth-squared values. We then apply a filter to this texture map, in this samples case a Gaussian blur so that the resulting texture stores average depth values hereby known as E(x) and average depth-squared values hereby known as E(x2). We can then use Chebychev's Inequality shown below to estimate the probability that a fragment with depth t will be in shadow.
