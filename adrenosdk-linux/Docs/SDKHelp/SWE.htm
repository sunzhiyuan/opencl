Shallow Water

This sample demonstrates a simple water simulation.

Details

This sample implements a liquid simulation as described in the 'Shallow Water Equations' chapter of the ACM Siggraph 2008 'Real Time Physics Class Notes' written by Nils Thuerey and Peter Hess. The liquid is modeled by three 2d arrays containing the height and x and y velocity values. The simulation operates entirely on the cpu. Once a frame needs to be generated, the program constructs a field of normals from the height values. From the normal field a texture is created that contains uv coordinates for lookup into a refraction and reflection image. The fragment shader simply uses these coordinates to read from the two image textures and combines the results.
