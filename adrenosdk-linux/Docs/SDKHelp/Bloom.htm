Bloom

This sample demonstrates how to create a bloom effect. Bloom effects are great ways to make scenes appear to be brighter than they are.

Details

This effect is achieved through the use of several off-screen passes. The first pass consists of the scene that we want to apply the bloom effect to. The second pass consists of the same scene with all of the objects rendered with a black constant shader. Next, we take this pass and apply a Gaussian blur to it. Finally, we add the results of this pass to our first pass which results in the background color bleeding into our objects.

There are a few additional tricks that can be applied to modify the appearance of the bloom effect as well as improve performance. The first trick is to down sample the FBO that is used for the second pass. This tends to increase the impact of the Gaussian blur while at the same time improving performance. Another trick that can be applied is to deform the surface of your objects that are rendered in the second pass along their normals. This enables us to sink the contours of the objects in a little bit which in turn increases the size of the bloom effect.

From left to right we have the first pass which consists of the scene without the bloom effect applied, then we have the second pass that consists of the scene rendered in black deformed inwards along the surface normals, followed by the same pass which has been down-sampled with a Gaussian blur applied, and finally we have the two passes added together.
