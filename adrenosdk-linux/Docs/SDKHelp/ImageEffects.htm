ImageEffects

This sample demonstrates how to achieve various post-processing effects.

Details

This sample demonstrates that OpenGL ES 2.0 can be used to achieve a variety of 2D effects that can be used for both gaming and non-gaming purposes alike.

BlurBlurs an image by averaging each pixel with its four diagonal neighbors.
EmbossMakes an image appear bumpy by subtracting the lower right three neighbors of each pixel from their upper left three neighbors.
LensAn image warping effect.
NoiseAdds 2D noise to an image.
SepiaTints an image to make it look like an antique photograph. The effect is achieved by calculating the luminance at a given pixel, and using the result as a lookup into a 1D sepia texture.
SharpenSharpens the image by increasing local contrast. This is done by adding the difference of each pixel and its four diagonal neighbors to each pixel.
SobelDetects edges in an image by computing the first order derivate of the image.
StaticAdds TV-like static to an image through the use of a 3D texture map of random values.
WarpWarps the center of an image with turbulence through the use of a 3D turbulent texture map.
