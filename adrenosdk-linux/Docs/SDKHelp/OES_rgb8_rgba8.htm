Overview

 

    This extension enables RGB8 and RGBA8 renderbuffer

    storage formats

New Tokens

 

    Accepted by the <internalformat> parameter of RenderbufferStorageOES:

 

        RGB8_OES             0x8051

        RGBA8_OES            0x8058
