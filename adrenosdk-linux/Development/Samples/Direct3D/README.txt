Building and distributing DirectX applications on Visual Studio 2012
QUALCOMM, Advanced Content Group - Snapdragon SDK
---------------------------------------------------------------------

1) You can build and run the samples for Windows at any time from within Visual Studio 2012

2) If you plan to run applications from outside Visual Studio 2012 you will need to create a package

3) In order to create a package you will need to get a Microsoft Developer License and setup Certificates

4) You will also need to change the Certificate information in the application's Package Manifest

5) You can find all the detailed steps from Microsoft Documentation
