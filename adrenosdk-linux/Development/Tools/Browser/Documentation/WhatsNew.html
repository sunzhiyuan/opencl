What's New in the Adreno SDK

Adreno SDK 3.8 includes a variety of new features.

New Features

Adreno SDK Emulator now supports Android Extension Pack OpenGL ES functions. Some of the key extensions include geometry and tessellation shaders.

This release includes a new OpenCL Debugger which allows source level debugging for OpenCL host and kernel programs on Snapdragon.

We have also added a new OpenCL developer guide that contains both general programming guidelines and optimzation techniques.

Texture Converter libraries built for Visual Studio 2013 have been added to the SDK.
